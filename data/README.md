# Dataset — Stack Overflow Developer Survey 2024

The dataset is **not included in this repository** because it exceeds GitHub's file size limit (~150 MB).

## How to download

1. Go to https://survey.stackoverflow.co/
2. Find the row for **2024**
3. Click on **"Data (CSV)"**
4. Download and unzip
5. Place `survey_results_public.csv` in this folder

## Files used in the analysis

- `survey_results_public.csv` — main dataset (65,437 rows × 114 columns)
- `survey_results_schema.csv` — column dictionary
